RTM駅メロMaker　使い方 ver 0.2
きのぴくんがわかるように書いたつもりなので、有識者の方は適宜読み飛ばしてくださいね。


＜ご利用の前に...＞
Python 3.9をダウンロード(開発は3.9.6なのでそれにすれば問題はないはず)
必ずPathは通すこと(microsoft storeからなら自動で通してくれるはずだし、webからのインストーラーにも途中でその項目があるはず)
コマンドプロンプトで"pip3 install pillow"もしくは"pip install pillow"と実行。（うまく動かない場合はバージョン9.1.1を指定する"pip3 install pillow==9.1.1"）
コマンドプロンプトで"pip3 install pygame"もしくは"pip install pygame"と実行。（うまく動かない場合はバージョン2.1.2を指定する"pip3 install pygame==2.1.2"）

＜起動方法＞
.pyファイルをダブルクリックしてくれれば起動するはずだけど、起動しなかったらidleから開いてF5キーで実行。exe化はする予定ないです。面倒だしウイルス判定されるから。
同一階層にbutton_.pngとEmaker_model.mqoとEmaker_texture.pngが必要。



＜「ファイル」タブについて＞
開く...プロジェクトのファイル（.rem）を開きます。このファイルの中身は単なるjsonです。
上書き保存...その名のとおりです。プロジェクトファイルを保存します。
名前を付けて保存...その名のとおりです。
RTM用に出力...RTMのmodsフォルダに入れるためのzip形式で出力します。これは、動画のエンコードのようなものですから、ある程度完成した後に行ってください。当然、このzipを直接編集することはできません。

＜メイン　ウィンドウについて＞

最上行............現在開いてるプロジェクトファイルの場所が表示されています。
その下の大きな四角形(ListBox)...ここには「追加」ボタンで追加した駅メロ一覧が表示されます。選択可能な項目は一つのみです。再生。削除のときに選択が必要になります。

社名/駅名など.......ここは省略可能です。何か書いた場合、サムネイルの左上に書かれます。文字を枠内に収めないと、サムネイル画像からはみでることがあります。「東横線　渋谷駅」とかいう使い方を想定しています。"メイリオ"のフォントにない文字(アラビア文字とか絵文字とか？)はたぶん文字化けします。知らんけど。この項目は、block idには関係ないので、重複しても構いません。

名前.............必須項目です。文字の種類・制約・数に関しては基本的には上と同様です。曲名を入れるのかなぁと想定しています。この項目は、block idには関係ないですが、アプリの都合上重複されると困ったことになるので重複禁止です。ただし、社名/駅名の欄が違えば大丈夫なはずです。

音声ファイル.........必須項目です。ogg形式のみのサポートです。他の形式はmodの制約上無理です。ファイル名に日本語とか空白を入れるとmodが正しく動かなそう。知らんけど。途中のディレクトリ名なら日本語入っててもかまいません。この項目は、block idには関係ないですが、重複していいかどうかは試してないのでわかりません。だれか人柱よろしく～

バツ印............空欄にします。あんま使わないかな。

「参照」ボタン........基本的に音声ファイルの指定はここからやってください。直接入力してもいいですが、エラーの原因になるので。

「追加」ボタン........名前および音声ファイルが指定されていれば、ListBoxに追加されます。夢中になりすぎて上書き保存をわすれることのないよう。

「削除」ボタン........ListBoxにて選択されているものを削除します。上書き変更はできませんので、ミスったら素直に削除してね～。

「再生」ボタン........ListBoxにて選択されているものを再生します。確認用です。
